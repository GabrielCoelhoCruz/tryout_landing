# Logos da Equipe SkyHigh AllStar

Esta pasta contém os logos oficiais da equipe.

## Logos disponíveis:

### 1. Logo Principal (Escudo)
- **arquivo:** `logo-shield.png` ou `logo-shield.svg`
- **Descrição:** Escudo com letras "S" e "H" entrelaçadas em laranja (#FF7F00), com borda laranja, fundo azul escuro (#0A1B4D) e raios elétricos azuis (#00BFFF)
- **Uso:** Logo principal no header, favicon, compartilhamento social
- **Formato recomendado:** SVG (escalável) ou PNG com fundo transparente
- **Tamanho:** Mínimo 512x512px para alta qualidade

### 2. Logo de Texto "SKY HIGH"
- **arquivo:** `logo-text.png` ou `logo-text.svg`
- **Descrição:** Texto "SKY HIGH" em laranja (#FF7F00) com borda metálica cinza
- **Uso:** Header alternativo, marca d'água, assinaturas
- **Formato recomendado:** SVG ou PNG com fundo transparente
- **Tamanho:** Largura mínima 800px para boa qualidade

### 3. Logo Escuro (para fundos claros)
- **arquivo:** `logo-shield-dark.png` ou `logo-shield-dark.svg`
- **Descrição:** Versão adaptada do logo para uso em fundos claros
- **Uso:** Footer, seções com fundo claro

### 4. Logo Claro (para fundos escuros)
- **arquivo:** `logo-shield-light.png` ou `logo-shield-light.svg`
- **Descrição:** Versão adaptada do logo para uso em fundos escuros
- **Uso:** Hero section, seções com fundo escuro

## Cores do Logo:

- **Laranja Primário:** `#FF7F00`
- **Azul Escuro:** `#0A1B4D`
- **Azul Elétrico:** `#00BFFF`
- **Azul Muito Escuro:** `#0A0A2A`

## Como adicionar os logos:

1. Coloque os arquivos de logo nesta pasta
2. Use os nomes sugeridos acima ou atualize as referências nos componentes
3. Para SVG: prefira formato vetorial para melhor qualidade em qualquer tamanho
4. Para PNG: use fundo transparente e resolução alta (mínimo 512x512px)

## Referências nos componentes:

Os logos são referenciados em:
- `src/components/Footer.tsx` - Logo no rodapé
- `src/app/layout.tsx` - Favicon e meta tags
- `src/components/SectionHero.tsx` - Logo no hero (opcional)
